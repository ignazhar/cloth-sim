# cloth-sim
Cloth simulation using C++ and SFML library.

### Required tools:
C++, SFML library, MINGW

### Windows Powershell execution:

Enter the src directory:

```PS ...> cd src```

Compile and execute:

```PS ...> mingw32-make```
